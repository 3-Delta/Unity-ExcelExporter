# Unity-ExcelExporter

1. Excel表格导出工具,支持导出json, lua, bytes, sqlite, protobuf, 原生代码
2. 支持Server/Client双端, 支持自定义类型, 支持枚举, 支持float, 支持表格间继承
3. 简单的外键表格检测
